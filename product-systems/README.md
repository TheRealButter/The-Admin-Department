# Product Systems

This folder contains the actual admin systems The Admin Department sells and installs.

These are not just service descriptions. Each system contains:

- workflow map
- tracker structure
- status rules
- SOP
- message/document templates
- weekly report format
- demo data
- implementation checklist

## MVP systems

1. **Invoice Admin System** — supplier invoices, receipts, proofs of payment, document filing, approvals, and exceptions.
2. **Sales Admin System** — enquiries, lead capture, response tracking, quote follow-up, and lost-lead prevention.
3. **Client Admin System** — onboarding, document collection, welcome process, folders, payment/request follow-ups, and onboarding reporting.

## Delivery rule

Every system must work manually before it is automated.

The first version should be installable using a tracker, templates, folders, and simple approval rules. Automation and AI can be added only after the workflow is proven.
