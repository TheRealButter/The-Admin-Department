# Gate 3 Readiness Checklist

Use this to decide whether a system is private-pilot ready.

## System

- [ ] Invoice Admin
- [ ] Sales Admin
- [ ] Client Admin

## Gate 2 proof

- [ ] Local workbook validation passed
- [ ] Google Sheets live test passed
- [ ] Expected pass/fail records confirmed
- [ ] Weekly report calculations confirmed
- [ ] Dropdowns and conditional formatting confirmed

## Private workspace

- [ ] Private workspace created
- [ ] Folder structure created
- [ ] Workbook copied into private workspace
- [ ] Demo data removed or clearly labelled
- [ ] Real data not stored in public repo

## Access control

- [ ] Sheet owner confirmed
- [ ] Editors confirmed
- [ ] Viewers confirmed
- [ ] Approval owners confirmed
- [ ] Weekly report recipients confirmed
- [ ] Access removal process confirmed

## Handover

- [ ] Handover test completed
- [ ] Tester processed clean record
- [ ] Tester processed missing-info record
- [ ] Tester processed blocked/risky record
- [ ] Tester explained weekly report
- [ ] Confusion points fixed or documented

## Pilot scope

- [ ] One system only
- [ ] One workflow only
- [ ] One report only
- [ ] Two-week pilot duration agreed
- [ ] Success/failure criteria agreed

## Privacy and risk

- [ ] No real client data in GitHub
- [ ] Sensitive documents kept in private workspace
- [ ] Human approval points documented
- [ ] No autonomous risky actions
- [ ] No payment/legal/accounting decisions automated

## Final decision

- [ ] Gate 3 passed — ready for controlled paid pilot
- [ ] Needs fixes before pilot
- [ ] Not ready
